# Bug Report Template

---

## Bug Information

| Field | Details |
|-------|---------|
| **Bug ID** | BUG-XXX |
| **Title** | Short and descriptive summary of the bug |
| **Date Reported** | YYYY-MM-DD |
| **Reporter** | Name of the person who found the bug |
| **Status** | New / In Progress / Fixed / Closed |

---

## Classification

| Field | Details |
|-------|---------|
| **Severity** | Critical / High / Medium / Low |
| **Priority** | High / Medium / Low |
| **Type** | Functional / UI / Performance / Security / Other |

> **Severity** measures the impact on the application.
> **Priority** measures how urgently the bug needs to be fixed.
