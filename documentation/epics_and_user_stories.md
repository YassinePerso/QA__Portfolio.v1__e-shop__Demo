## EPIC-01 : User Account Management

### US-01 : Account Registration
**As a** visitor
**I want to** create a new account
**So that** i can log into the website as a registered visitor

#### Acceptance Criteria
- [ ] Email must be in valid format (e.g. user@example.com)
- [ ] Password must be at least 6 characters
- [ ] Email must not already be registered
- [ ] After successful registration, user is redirected to home page as a logged-in user
- [ ] If registration fails, a clear error message is displayed
